Descripción

El objetivo de este sitio es servir de respaldo para aquellas ocasiones en las que el servidor de EpubLibre se encuentra caído o sobrecargado. Los libros y enlaces que componen la siguiente base de datos pertenecen al sitio de EpubLibre y son actualizados de manera regular.


Guía de uso:

Existen dos opciones para acceder a la base de datos. Por un lado se puede descargar haciendo click aquí . Se descargará un archivo .csv (se abre con excel) que contiene el listado de libros y sus respectivos enlaces para descargarlos por torrent. Una vez encontrado el libro deseado simplemente copiar y pegar en enlace en la barra de direcciones del navegador.

La otra opción es consultar el listado aquí abajo. Para ello se debe ingresar dónde dice epublibre. A continuación aparecerán 3 casilleros. En el primero (-column-) se debe ingresar la columna en la cuál se desea realizar la búsqueda. En el segundo conviene elegir la opción -contains-. Y en el último casillero se debe introducir el término que sea desea buscar. Finalmente se realiza click en Apply y la base de datos mostrará los resultados disponibles. Al igual que en el método anterior, una vez encontrado el libro deseado simplemente se debe copiar y pegar en la barra de direcciones el texto que figura en la columna "enlace".
